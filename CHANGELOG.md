# Changelog

## 0.1.0 · 2026-07-22

### Toegevoegd

* Moderne Windows-interface met statusweergave
* Eénklik-reset van de Bluetooth-adapter
* Losse aan/uit-schakelaar
* Automatische aanvraag van beheerdersrechten
* Adapterdetectie met voorkeur voor Intel Wireless Bluetooth
* Logbestand voor acties en fouten
* Windows-launcher en PyInstaller-buildscript
* Automatische tests en GitHub Actions

### Handmatige controle

Test de eerste keer op de doelcomputer om te bevestigen dat Windows de adapter als `Intel(R) Wireless Bluetooth(R)` herkent.
